# FASHION-MNIST: desarrollo de un modelo MLP

En el presente informe .ipynb buscamos como grupo trabajar el desarrollo de una red neuronal para resolver un problema de clasificación de imágenes, en este caso, con FASHION-MNIST. FASHION-MNIST es un dataset que contiene 60.000 imágenes de 10 tipos de prendas diferentes (camisas, pantalones, abrigos, etc.) en escala de grises y de 28x28 pixeles cada una.

## Objetivo de este proyecto
A lo largo de este proyecto lo que buscamos desarrollar es un modelo que sea capaz de identificar patrones de datos etiquetados y clasificar correctamente imágenes nuevas. Para lograr este objetivo, se crea un modelo de red neuronal que presente normalización, transformación, optimización y técnicas de regularización para que sea robusto y estable, capaz de predecir correctamente las prendas que se presenten a futuro.

## Evaluación y seguimiento de métricas
Además de incluir todo aquello que involucre la creación del modelo, también se buscó hacer multiples evaluaciones durante su desarrollo paso a paso para seguir una ruta que permita identificar las mejores configuraciones sobre los datos, en conjunto con la utilización de métricas para medir el rendimiento de los modelos hasta llegar al mejor modelo definido.

# Instrucciones de ejecución

El documento .ipynb adjunto ha de ser ejecutado en un entorno capaz de soportar el uso de redes neuronales. En este caso, se recomienda hacerlo a través de la plataforma "Colab" proporcionada por Google y cambiar las configuraciones del acelerador por hardware a "GPU T4", el cual se encuentra dentro del menú de cambios del entorno de ejecución asociado.

Tras aplicar los cambios sobre el entorno, puede abrir el cuaderno y conectarse al entorno alojado en la esquina superior derecha. Este informe sigue un orden secuencial, por lo que debe de ser ejecutado en el orden base en el que se encuentra proporcionado en este repositorio.
